### Running Locally

To run the project on your local machine:

1. Clone the project:
   ```sh
   git clone https://github.com/fac30/digital-fitness-vicky-emil.git
   ```

2. Navigate to the project directory:
    ```sh
    cd digital-fitness
    ```

3. Open the project in your browser:
    ```sh
    open index.html
    ```
